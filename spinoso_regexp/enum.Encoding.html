Enum spinoso_regexp::Encoding

source · 
pub enum Encoding {
    Fixed,
    No,
    None,
}
Expand description

The encoding of a Regexp literal.

Regexps are assumed to use the source encoding but literals may override the encoding with a Regexp modifier.

See Regexp encoding.
